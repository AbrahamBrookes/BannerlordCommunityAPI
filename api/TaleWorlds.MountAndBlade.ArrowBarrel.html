Class ArrowBarrel

The ArrowBarrel is a good example of an object that exists in the mission world (not the map view) that inherits a lot of different things.

It's inheritance tree includes SynchedMissionObject which means it can be managed in multiplayer games.

it is a ScriptComponentBehaviour which means it can be attached as a component to a GameEntity.
