Class UsableMachine

UsableMachine is a base class for multiplayer siege equipment and other "useable" props, specifically in the multiplayer mission/battle view(not the map view).

"Base class" means it is designed to be extended in order for devs to create new mission objects. It is marked abstract to prevent people from trying to create a new instance of this class directly.

A good example of how this is used is the ArrowBarrel class.
