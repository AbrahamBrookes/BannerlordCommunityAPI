Class GameEntity

The GameEntity class is the base of a lot of other game objects. It is conceptually similar to Unreal Enginges Actor class, or Unity's GameObject class.

GameEntity is a sealed class, which means it cannot be extended (aka "Inherited from").

You can modify the behaviour of a GameEntity by attaching a ScriptComponentBehaviour.

Classes that extend ScriptComponentBehaviour have access to their attached GameEntity, and you can see more in the ScriptComponentBehaviour documentation
